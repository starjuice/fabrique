# Fabrique

Factory support library for adapting existing modules for injection as dependencies.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'fabrique'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fabrique
